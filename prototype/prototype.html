主要功能特色

YouTube MindMap 提供多種強大功能，幫助你快速消化和整理影片內容，提升學習效率和知識吸收。

自動擷取字幕

系統自動從 YouTube 影片中提取完整字幕，支援多語言影片，無需手動輸入或複製貼上。

AI 內容摘要

運用 Deepseek API 先進的自然語言處理技術，智能分析影片內容，提取關鍵點並生成結構化大綱。

互動式思維導圖

將 AI 生成的大綱轉換為視覺化思維導圖，支援展開、收合、縮放等互動功能，幫助你理解內容層次。

多種匯出格式

支援將思維導圖匯出為 Markdown、PNG、SVG 等多種格式，方便保存、分享和用於其他筆記系統。

歷史記錄管理

自動儲存所有已處理的影片及其思維導圖，方便隨時查看和重新訪問，無需重複處理相同影片。

自訂摘要參數

可調整 AI 摘要的詳細程度、關注點和風格，滿足不同學習者和不同類型內容的需求。

使用流程

YouTube MindMap 的使用過程簡單直覺，只需三個步驟，即可將冗長影片轉換為清晰的思維導圖。

常見問題

我們整理了用戶最常提問的問題，幫助您更好地了解 YouTube MindMap 的功能和使用方式。

YouTube MindMap 支援哪些語言的影片？

我們的系統支援多種語言的影片處理，包括中文（繁體和簡體）、英文、日文、韓文、法文、德文、西班牙文等常見語言。系統會自動識別影片語言，並提供相應的字幕提取和摘要生成。對於非主流語言，我們也在不斷擴充支援範圍。

如何匯出和分享生成的思維導圖？

生成思維導圖後，您可以點擊界面上的「匯出」按鈕，選擇您需要的格式，包括 Markdown、PNG、SVG、PDF 等。高級用戶還可以匯出為 PPT 格式。匯出後，您可以下載文件保存，或使用「分享」功能獲取一個可公開訪問的連結，方便分享給同事、朋友或學生。

免費版本有什麼限制？

免費版本每月可處理 5 次影片，每個影片時長不超過 15 分鐘，且僅保留最近 7 天的歷史記錄。同時，免費版僅支援基本的思維導圖功能和 Markdown 格式匯出。高級功能如自訂摘要參數、多種匯出格式、更長影片處理等需要升級到付費方案。

系統如何保證摘要和思維導圖的質量？

我們採用 Deepseek API 領先的自然語言處理技術，結合自主開發的內容結構化算法，確保生成的摘要準確反映影片核心內容。此外，我們還針對不同類型的影片（教育、演講、教程等）優化了處理流程，提高摘要質量。用戶還可以通過調整摘要參數，如細節程度、關注點等，進一步優化結果。

如果影片沒有字幕怎麼辦？

對於沒有官方字幕的影片，我們會嘗試使用語音識別技術自動生成字幕。這一功能在大多數語言中都有不錯的效果，但準確性可能受影片音質、背景噪音和口音等因素影響。對於自動生成的字幕，系統會在處理過程中標記，並在摘要結果中提示用戶。
